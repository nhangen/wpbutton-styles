A simple WordPress plugin to enable the WP Core UI button classes on the front-end of your website.

== Description ==

A simple WordPress plugin to enable the WP Core UI button classes on the front-end of your website. Includes support for submit inputs, .button, and .button-primary.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

Upload `wordpress-buttons.php` to the `/wp-content/plugins/` directory

== Or ==

Activate the plugin through the 'Plugins' menu in WordPress
